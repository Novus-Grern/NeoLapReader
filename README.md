# NeoLapReader
A python file that takes a csv of lap times and displays them as a time graph

## What it do?
takes a csv or log file from an assetto corsa plugin (laplogger) and can display a time graph along with other neat things

## How it do?
Takes an excel file generated by a certain lap logging plugin
